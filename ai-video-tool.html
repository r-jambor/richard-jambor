Built to automate a fragmented process

Modern AI video creation typically requires switching between multiple disconnected systems — script generation, text-to-speech, image/video models, and manual timeline assembly in a video editor. This project was built to explore whether these steps can be unified into a single automated execution pipeline.

Problem

AI content generation is fragmented across multiple independent tools and manual steps.

Solution

A modular orchestration pipeline that coordinates scripting, voice generation, media creation and final assembly into one repeatable process.
